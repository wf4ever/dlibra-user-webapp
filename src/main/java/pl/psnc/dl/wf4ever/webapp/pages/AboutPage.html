What is this service

Wf4Ever User Management Service allows users to create accounts in dLibra and give access permissions to other applications. Additionally, users that already have accounts in myExperiment can import their data to dLibra.

Other Wf4Ever applications

At the moment of writing this, the only application allowing to view the imported content is dLibra Editor Application (check step 2 in here for details on how to use it). The ROBox currently creates its own accounts in dLibra but it should connect to accounts created by this service soon. A command line interface for interacting with dLibra is also under development.
