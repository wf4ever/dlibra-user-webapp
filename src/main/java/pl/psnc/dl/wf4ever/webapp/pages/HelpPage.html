What is the goal of this service?

This service is the only place in Wf4Ever in which you can create an account in dLibra. If you don't have an account in dLibra, it is created automatically for you when you log in.

Other Wf4Ever services can use this service for user authentication and to ask for permission to change user data in dLibra.

What is OpenID?

OpenID is a method of authenticating users using their accounts in other Internet services, such as Google or Yahoo.
