This wizard allows you to copy your resources (workflows/files/packs) from myExperiment to dLibra, aggregating them as Research Objects (RO). The wizard will help you to select how you want to aggregate these resources.

Since the actual structure of RO and its metadata is still under development, for every imported resource its accompanying RDF file will also be transfered to make sure that the metadata will be available for processing in the future.

The wizard has the following steps:

  1. Select resources. Choose if you want to import all resources as 1 RO or every resource as a separate RO.
  2. Choose RO names.
  3. Go back to step 1 if you want to define more ROs.
  4. Create ROs in dLibra, copy selected resources from myExperiment to dLibra.
  5. Summary.
